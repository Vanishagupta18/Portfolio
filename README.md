# 💻 IDE Portfolio — VSCode Themed Developer Portfolio

A modern, interactive developer portfolio built with a **VSCode-inspired UI**, designed to showcase projects, skills, and experience in a unique and engaging way.

---

## 🚀 Tech Stack

- **Frontend:** React.js, Vite  
- **Styling:** Tailwind CSS, Custom CSS  
- **State Management:** React Context API  
- **Routing:** React Router  
- **Other Libraries:** React Icons, GitHub integrations  

---

## ✨ Features

- 🧠 VSCode-inspired layout (Explorer, Tabs, Sidebar)
- 📂 Dynamic file explorer with clickable sections
- 🖥️ Multi-tab content system (like a real IDE)
- 🎨 Dark theme UI with developer-focused design
- 📊 Structured data-driven content (projects, skills, contact)
- 🔗 Integrated social links (GitHub, LinkedIn, LeetCode)
- ⚡ Fast performance using Vite
